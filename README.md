# cloud-music

### 项目简介
本项目是用Vue-cli3框架搭建的仿网易云音乐Vue网页版

### 技术栈
Vue、VueX、Vue Router、axios等

### 实现的功能
音乐播放、暂停、更换主题色、推荐页、歌单页、歌单详情页、排行榜页、歌手页、歌手详情页、专辑详情页、播放页等
